# Copy Style Guide

## Idioma
Toda la app debe usar español neutro.

## Regla obligatoria
No usar voseo argentino o rioplatense en ninguna parte de la app.

## No usar
- querés
- podés
- tenés
- decidís
- aprobás
- descartás
- mandame
- decime
- hacé
- revisá
- ingresá
- probá
- elegí
- mové
- tocá

## Usar en cambio
- quieres
- puedes
- tienes
- decides
- apruebas
- descartas
- envíame
- dime
- haz
- revisa
- ingresa
- prueba
- elige
- mueve
- toca

## Tono
- claro
- natural
- moderno
- profesional
- cercano
- sin modismos argentinos
- sin voseo
- sin tono infantil
- consistente en toda la app

## Aplicación
Estas reglas aplican a:
- textos de UI
- onboarding
- tooltips
- bandeja de Nova
- notificaciones
- modales
- mensajes de error
- prompts visibles
- landing page
- ejemplos
- textos generados por IA dentro del producto
